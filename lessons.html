XHTML

-XHTML is an acronym for eXtensible HyperText Markup Language.
-The ”x” came from XML or eXtensible Mark-up Language. -Though different from HTML, XHTML also uses the filename extension.html
-The content of the actual coding might change because of XHTML conventions but it still holds the same filename extension.
-XHTML is stricter and cleaner version of HTML.
-Aimed to replace HTML.
-All new Web browsers support XHTML, because there came to a point that a huge amount of web pages have been written in the wrong syntax. Since web browsers still force to interpret HTML the way they could, sometimes the wrong syntax
-could still display a logically correct output.
